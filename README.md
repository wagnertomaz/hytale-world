# Hytale Bug Report: Transparency & OpenGL State Corruption

This repository contains the world save and logs regarding a critical rendering issue where game assets and world chunks become transparent, likely due to an OpenGL state corruption as identified by @Slikey.

## Issue Description
- **Bug:** Visual transparency occurring during biome transitions (Swamp) or time-of-day changes (Night), leaving only a few background blocks visible.

*Detailed hardware profile from AMD Software: Adrenalin Edition:*

- **CPU:** AMD Ryzen 5 5600G (6 Cores)
- **GPU:** AMD Radeon(TM) Graphics (Integrated Vega 7)
- **VRAM:** 2048 MB DDR4
- **RAM:** 16 GB
- **Driver Version:** 23.11.1 (Released: 10/17/2023)
- **OS:** Windows 11
